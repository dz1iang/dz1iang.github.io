hexo搭建博客

发表于 2019-06-11 | 分类于 博客编辑 | 阅读数

使用hexo可以快速搭建个人网站博客，hexo提供了方便的命令行进行操作，使用的是markdown编辑风格。同时，可以设置个人的博客主题风格结构等等。也可以与github绑定，发布到github.io。

快速开始

  • git安装mac系统中Xcode Command Line Tools已经包含，或者在终端输入git即提示安装。

  • node环境使用nvm：https://github.com/nvm-sh/nvm

  • 安装hexo

初始化目录

  • hexo初始化博客

  • 浏览器打开即可看到本地运行的服务：http://localhost:4000

修改next主题

  • 查看当前支持主题,目录下只有一个主题

  • 查看配置的主题，初始化配置的是，theme: landscape

  • 下载next

  • 修改主题配置为，theme: next

  • 浏览器再次访问即可发现默认主题已经修改为next：http://localhost:4000/

创建新文章

  • hexo创建文章

  • 浏览器再次访问首页已经展示了新文章：http://localhost:4000/

发布github博客

  • github创建io项目，命名格式遵守：账户名.github.io，勾选：Initialize this repository with a README ，然后创建。

  • github设置ssh参考：https://help.github.com/en/articles/connecting-to-github-with-ssh

  • hexo配置github，找到deploy配置，修改repository为刚才创建的io项目地址

  • hexo发布到github

  • 等待一小会，浏览器访问github地址即可看到博客：
